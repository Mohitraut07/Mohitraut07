Trending Repos I explored

25 Sept 2023

I recently explored few repositories on Github , among them the trending repository which I explored was PowerShell.It had few new commits after years of time.The commit messages were classified according to the contribution performed.The repo had around 61k stars and around 1k contributers.
The README file was designed so well that it replaced any need for official documentation.The whole process on how to access the project was explained so that even a layman can understand it.
One key aspect to note is that the branch names were written in a professional manner.It not only included contribution specific name but also the current version .
I very much enjoyed exploring the repo and learned quite a lot of things.Overall it was an marvelous experience.

What is hacktoberfest and why you should care?

25 Sept 2023

Hacktoberfest as the name suggests is an event organised by Github and Digital Ocean . In this fest developers have to contribute to open source projects by firstly signing up on hacktoberfest.com then contributing to repos to resolve issues labeled under the fest.
One can participate any time between 1 October to 31 October. For contributing in open source dev's are gifted with equivalent prizes and merchandises.
Contributing to open source is very good for an aspiring developer and can be great for carrer advancement.

-Mohit Raut
